标题

HTML 标题（Heading）是通过<h1> - <h6> 标签来定义的。 标题独占一行，并且和下一行留有间距，h1最大，h6最小。 <h1>这是一个标题</h1> <h2>这是一个标题</h2>

段落

HTML 段落是通过标签 <p> 来定义的。 段落独占一行。 <p>这是一个段落</p>

换行

<br> 元素在文本中生成一个换行（回车）符号。 <p>我喜欢你<br> 也喜欢他 </p>

链接

HTML 链接是通过标签 <a> 来定义的。 可以通过它的href属性创建通向其他网页、文件、电子邮件地址、同一页面内的位置或任何其他 URL 的超链接。 如果存在 href 属性，当 <a> 元素聚焦时按下回车键就会激活它。 属性： href 超链接所指向的 URL。 download 导致浏览器将链接的 URL 视为下载资源。 filename：决定文件名的值。/ 和 \ 被转化为下划线（_）。 target 该属性指定在何处显示链接的URL _self：当前页面加载。（默认） _blank：在新标签页打开。 _parent：在父框架中打开链接文档。 _top：在窗口的整个主体中打开链接的文档。 framename：在指定的 iframe 中打开链接文档。 <a href="URL" target="_blank">这是一个链接</a> <a href="#id">跳转到相同页面的id位置</a>

图像

<img>元素将一张图像嵌入文档。 src 属性是必须的，它包含了你想嵌入的图片的路径。 alt 属性是对图像的文本描述，如果由于某种原因无法加载图像，浏览器也会在页面上显示 alt 属性中的备用文本
